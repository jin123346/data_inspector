# Data Inspector (Image Inspector)

관찰·표본 데이터와 이미지 자료를 효율적으로 검수하고 정리하기 위한  
**로컬 기반 데이터 검수/엑셀 템플릿 생성 도구**입니다.

본 프로젝트는 서버 없이 사용자 PC에서 실행되는 Electron 애플리케이션으로,  
엑셀 템플릿 생성, 이미지–데이터 검수, 위치정보(지도/좌표) 연계를 목표로 합니다.

---

## 📌 프로젝트 배경

관찰·표본 데이터는 보통 엑셀로 관리되고,  
이미지는 별도의 폴더에 누적되는 경우가 많아 다음과 같은 문제가 발생합니다.

- 엑셀 데이터와 이미지 파일의 **일치 여부를 수작업으로 확인**
- 이미지 파일명 규칙 불일치
- 표본/관찰 DB 번호 수동 생성으로 인한 오류
- 작업 중 중단 시 데이터 유실 위험

이 프로젝트는 이러한 문제를 **로컬 환경에서 안전하게 해결**하기 위해 시작되었습니다.

---

## 🎯 프로젝트 목표

- 관찰정보 / 표본정보 중 **유형을 먼저 선택**하여 엑셀 템플릿 생성
- 표준화된 컬럼 구조를 가진 **엑셀 템플릿 자동 생성**
- 규칙 기반 **표본 DB 번호 자동 생성**
- 이미지 미리보기 기반 데이터 검수 UI 제공
- 지도 클릭을 통한 **위치 → 위도·경도·주소 자동 획득**
- 서버 없이 실행 가능한 **로컬 데스크톱 도구**

---

## 🛠 기술 스택

- **Electron**: 데스크톱 애플리케이션 프레임워크
- **Node.js**: 로컬 파일 시스템 접근, 엑셀 생성 로직
- **HTML / CSS / JavaScript**: UI 구성
- (예정)
  - exceljs 또는 xlsx: 엑셀 템플릿 생성
  - Leaflet + OpenStreetMap: 지도 UI
  - Nominatim API: 좌표 ↔ 주소 변환
